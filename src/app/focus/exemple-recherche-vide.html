Démo de directive manipulant le DOM

La directive setFocus permet de mettre le focus sur un champ selon certaines conditions. Ici, on met le focus sur le champ Nom si aucun nom ou code postal n'a été saisi au chargement de la page (ces informations étant liées dans les bindings), sinon on met le focus sur le champ X.
